### Just a template generator base on Pug :P
- 1. load your template.pug inside 'templates/' dir.
- 2. load your data.json inside 'data/' dir. (with the same name as the template.pug)
- 3. 'npm run start' to generate your html templates based on your json data
- [EXTRA] 4. you can load an additional x.base-data.json inside 'data/' dir, to add these data to every template (this point will be deprecated because fa cagare ma mi serviva per il fabulous S&Y)
